# Surfgame 3D Base

Starter scene for `Twopants24/surfgame`.

## Includes

- Three.js browser bootstrap
- Animated ocean surface
- Surfboard and rider placeholder model
- Keyboard movement and boost
- Jump / hop input
- Follow camera
- Island and buoy set dressing

## Run

Open [index.html](/Users/logan/Documents/New%20Project/surfgame/index.html) in a browser.

## Controls

- `WASD` or arrow keys: move and steer
- `Shift`: boost
- `Space`: hop
